O que é a Tecnologia Java e porque preciso dela?

Java é uma linguagem de programação e plataforma computacional lançada pela primeira
vez pela Sun Microsystems em 1995. Existem muitas aplicações e sites que não funcionarão,
a menos que você tenha o Java instalado, e mais desses são criados todos os dias.
O Java é rápido, seguro e confiável. De laptops a datacenters, consoles de games
a supercomputadores científicos, telefones celulares à Internet, o Java está em todos os lugares!
